获数千万元B1轮融资，用DevOps提效运维行业

xiaomengxiao 2018-07-11 156 次 收藏 0

今日，优维科技（UWinTech）对外宣布完成数千万元B1轮融资，由斯道资本领投，老股东祥峰投资参与跟投。此前于2016年11月获得过3000万元的A轮融资，由祥峰投资领投，君盛资本跟投。此轮融资将主要用于新一代Devops及运维平台的建设。

36氪曾报道，优维科技定位一站式DevOps及运维解决方案服务商。

创始人兼CEO王津银认为，随着底层硬件云化的普及，越来越少的企业会自己拥有机房，企业自己做硬件维护的需求在缩水。与此同时，上层应用变多，加之，用户需求多变需要企业产品快速迭代，所以，应用、系统方面的运维会变得更加重要。

优维科技的核心产品是EasyOps，EasyOps基于DevOps理念，面向应用提供全生命周期管理（ALM），主要提供应用CMDB，持续交付，运维自动化，立体化智能监控和IT运营分析五大服务模块。

从监控开始，优维先将基于应用的监控体系进行分层，构建端到端的能力，包括底层的基础设施、上层的应用组件、组件中的服务、组件里的日志、用户册的监控5层。这里监控采集到的有结构化数据和非结构化数据，之后通过大数据实时处理技术，再结合基于应用的拓普视图、关联关系等，来进行溯源、分析、告警、决策判断。例如，当发现服务器容量过高时，可以通过自动化处理平台添加一台服务器，自动扩容、降低负载。

至于是否可以规模化复制、通过标准化产品适配企业，王津银表示：“IT管理的核心可以抽象成3件事情——终端执行命令、传输文件、采集数据。不同的应用，就是这三件事情的逻辑拼凑不同，想要完全标准化跑通所有企业肯定不现实，但是可以从底层出发，做到适配同一个行业。”

还有企业关心的数据安全问题，优维科技主要在为企业做私有化部署。

此外，在具体销售过程中也存在障碍，运维人员往往会抵触产品，认为企业购买产品会取代自己的工作。王津银坦言，这的确是实际中的难点，他们的用户画像分两种：一类是传统企业，像证券、运营商、 电信、保险等，由于用户需求激增，传统的人力运维不能满足企业需求，所以必须要辅以产品来实施。另一类是互联网＋企业，跟这部分客户在交流时，强调的不是角色替代，而是传递管理的价值，体现产品对于场景问题的解决效率。

据悉，优维科技成立于2015年，总部位于深圳，目前在广州、武汉设有两个研发中心，并在北京，上海设有分公司及本地化服务团队。客户覆盖金融、航空、通信、物流等领域，包括招商银行、民生银行、上汽通用，国信证券、春秋航空、中国移动、国家电网、德邦物流等。
